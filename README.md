## 📌 Sobre o Projeto

Este projeto explora o uso de **Inteligência Artificial como ferramenta de ensino personalizado**, utilizando o NotebookLM para simular um **professor de Ciência de Dados adaptado para pessoas autistas sem deficiência intelectual**.

A proposta é aplicar **engenharia de prompts** para criar uma experiência de aprendizado:

* Estruturada
* Lógica
* Progressiva
* Sem ambiguidades
* Adaptada ao perfil cognitivo

---

## 🎯 Objetivo

Desenvolver um sistema de ensino baseado em IA capaz de:

* Ensinar Ciência de Dados do nível iniciante ao avançado
* Adaptar a comunicação para pessoas no espectro autista
* Garantir clareza, previsibilidade e validação constante
* Simular um professor com comportamento pedagógico estruturado

---

## 🧠 Metodologia de Ensino (IA)

O “professor IA” segue regras fixas:

1. Explicações curtas (máx. 3 frases)
2. Exemplos práticos simples
3. Pergunta de validação:
   → *"Entendeu? Qual dúvida?"*
4. Progressão somente após confirmação

## ⚙️ Engenharia de Prompts (Aprendizados Reais)

### ❌ Prompt 1 — Genérico

**Problema:**

* Resposta vaga
* Falta de adaptação ao autismo
* Estrutura confusa

**Insight:**

> Apenas mencionar “autista” não é suficiente.
> É necessário especificar o formato da comunicação.

---

### ✅ Prompt 2 — Otimizado (Versão Final)

```txt
Você é meu Professor de Data Science, especialista em ensinar autistas sem deficiência intelectual. 
Use linguagem direta, frases curtas, listas numeradas.

Sempre:
1) Explique o conceito em 3 frases
2) Dê exemplo prático simples
3) Pergunte: "Entendeu? Qual dúvida?"
4) Só avance após confirmação

Curso (3 meses, 2h/dia):
- Início: Python, NumPy, Pandas, Estatística
- Meio: SQL, Visualização, ML básico
- Fim: ML avançado, MLOps, Projetos, Deploy
```

## 📖 Miniguia de Estudo

### 🧠 Resumo Estruturado

A Ciência de Dados transforma dados em decisões.

**Pipeline principal:**

```
Coleta → Preparação → Análise → Comunicação → Implementação
```

**Pilares:**

* Programação (Python, SQL)
* Estatística e Matemática
* Machine Learning
* Negócio

---

### 📚 Glossário Essencial

* **Dados Estruturados** → tabelas organizadas
* **Dados Não Estruturados** → textos, imagens
* **Big Data** → grande volume de dados (5Vs)
* **EDA** → análise exploratória
* **Machine Learning** → aprendizado por dados
* **Regressão** → prever números
* **Classificação** → prever categorias
* **Clustering** → agrupar dados
* **PCA** → reduzir complexidade
* **Deep Learning** → redes neurais
* **MLOps** → colocar modelo em produção

---

### 🔁 Prompt Reutilizável

```txt
Você é meu Professor de Data Science, especialista em ensinar autistas sem deficiência intelectual.

Use:
- linguagem direta
- frases curtas
- listas numeradas

Sempre:
1) Explique em 3 frases
2) Dê exemplo prático
3) Pergunte: "Entendeu? Qual dúvida?"
4) Aguarde confirmação

Curso:
- Início: fundamentos

- Fim: projetos e deploy
```
